Wilkommen auf der BungeeUtil Webseite

BungeeUtil ist eine Erweiterung für den Proxy BungeeCord. Dieses Plugin bietet außer eine PacketLib weiter umfangreiche API´s:
(Inventory API,Particel API,NPC API,NBTTag API API,DataWatcher API,ChatLib).
